    # Banners

    A banner displays a prominent message and related optional actions. Banners can be used to notify users in a less disruptive way than Dialogs. Also consider Snackbars for alerts that disappear automatically.

    You will need to sign in to save changes.
    Continue as Guest Sign In

    The above banner is sticky (scroll down).

    This is a banner without actions. You have lost connection to the internet. This app is offline.
    Dismiss Try again
     It is now safe to turn off your computer.
    Shut down
     Reset Banners 
    <mx-banner :is-open="isOpenA" sticky class="top-56">
  You will need to sign in to save changes.
  <div slot="actions">
    <mx-button btn-type="text" @click="isOpenA = false">Continue as Guest</mx-button>
    <mx-button btn-type="text" @click="isOpenA = false">Sign In</mx-button>
  </div>
</mx-banner>

    # Properties

    Property Attribute Description Type Default
    error error boolean false
    isOpen is-open Toggles the banner with a transition. boolean false
    sticky sticky When set, position: sticky will be applied to the banner. boolean false
