# Colors

Class Description Default
.text-primary Dark Text Color #111827
.text-primary-inverted Light Text Color #fff
.bg-primary Light Background Color #e1f5ff
.bg-primary-inverted Dark Background Color #0457af
.bg-gray Light Gray Background #fafafa
.bg-gray-inverted Dark Gray Background #e8e8e8

# Variables

The above classes are defaulted. These values can be overridden by setting a color on a CSS Variable.

:root {
  /* Global Utility Variables */

  --mds-bg-primary-inverted: #0457af;
  --mds-bg-primary: #e1f5ff;
  --mds-bg-secondary-inverted: #edfffc;
  --mds-bg-secondary: #00696b;
  --mds-bg-gray-inverted: #e8e8e8;
  --mds-bg-gray: #fafafa;

  --mds-text-primary: #111827;
  --mds-text-primary-inverted: #fff;

  /* Component Variables */

  /* #region buttons */
